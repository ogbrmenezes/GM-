# 👨‍💻 Sobre mim

Sou um profissional de tecnologia com experiência em **suporte técnico N1, N2 e N3**, automação de processos e desenvolvimento de soluções para ambientes corporativos.

Atuei no suporte ao ambiente tecnológico da **Americanas**, focado em resolução de incidentes, estabilidade de sistemas e melhoria operacional.

Atualmente atuo como **Solutions Architect**, trabalhando na criação e estruturação de soluções tecnológicas que conectam necessidades de negócio com implementação técnica.

Também desenvolvo projetos pessoais envolvendo **frontend web e automação com Python**, aplicando tecnologia para resolver problemas reais.
